## Historical Engineering Records

The files in [`history/`](history/) preserve the design phases, diagnostic
spike results, and validation evidence that led to the current implementation.
They are useful context, but unchecked items and branch references are not the
current release status or contributor workflow.

Current user-facing behavior is defined by the root [README](../README.md), the
maintained guides above, and the source code.
